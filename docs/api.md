# FinAI Nexus API Documentation

## GraphQL Endpoints
- `/health`: Health check
- `/portfolio`: Portfolio management
- `/trade`: Trade execution
- `/compliance`: Regulatory monitoring
- `/fraud`: Fraud detection

## REST Endpoints
- `/api/v1/portfolio`
- `/api/v1/trade`
- `/api/v1/compliance`
- `/api/v1/fraud`

See Postman collection for request/response examples.
